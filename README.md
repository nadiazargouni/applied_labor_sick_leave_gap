# Applied Labour Economics: Sick leave gap between migrant and native workers
This repository contains the main scripts for the Applied Labour Economics report of Haytham Tahriny and Nadia Zargouni. 

The files can also be found in the CASD project ENSAE05, in `Nadia_Haytham/R Codes/`. 

The numbering of the files corresponds to the logical order of the analysis: one file might call precedent files to fetch data, they are to be ran in the order of the numbering. 

In  `Nadia_Haytham/Outputs`, the outputs of the code can be found. Each R script which generates outputs such as figures or tables is associated to a subfolder corresponding to its numbering: III_03 will generate figures stored in `Nadia_Haytham/Outputs/III_03`. 

*Important*: codes cannot be ran as is without  `Nadia_Haytham/R Codes/auxiliary` R scripts which are auxiliary functions to format tables or to fetch data from raw sources using DuckDB. 

Necessary packages are specified in the first script.
